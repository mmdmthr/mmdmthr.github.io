Setting Keyboard Keychron K2 di Ubuntu

Mengatur Tombol Fn + F

Keyboard Keychron di Linux menggunakan driverÂ hid_appleÂ (bahkan dalam mode Windows/Android), baik dalam mode Bluetooth maupun Kabel. Secara default, driver ini menggunakan tombol F (F1-F12) sebagai tombol pintasan multimedia dan kita harus menekan tombolÂ fnÂ + untuk mendapatkan tombol F1 hingga F12 yang biasa.

Untuk mengubahnya, kita perlu mengubah parameterÂ fnmodeÂ untuk modul kernelÂ hid_appleÂ . Berikut beberapaÂ dokumentasiÂ di dalamnya, tetapi ringkasan singkatnya dapat ditemukan di bawah:

  • 0 =Â disabled: Menonaktifkan tombol fn. Menekan fn+F8 akan berperilaku seperti kita hanya menekan F8
  • 1 =Â fkeyslast: Tombol fungsi digunakan sebagai tombol terakhir. Menekan tombol F8 akan bertindak sebagai tombol khusus (pintasan). Menekan fn+F8 akan berperilaku seperti F8
  • 2 =Â fkeysfirst: Tombol fungsi digunakan sebagai tombol pertama. Menekan tombol F8 akan berperilaku seperti F8. Menekan fn+F8 akan bertindak sebagai tombol khusus (play/pause).

Uji bagaimana keyboard berperilaku setelah mengubah nilai. Pilih salah satu yang sesuai kebutuhan untuk kita. Setelah kita menemukan nilai yang sesuai untuk kita, kita dapat membuat perubahan permanen:

  1. Buat fileÂ /etc/modprobe.d/hid_apple.conf
  2. Tambahkan baris ini di dalam file:Â options hid_apple fnmode=<value>, ganti <value>Â dengan nilai yang kita inginkan pada tahap sebelumnya (0, 1 atau 2)
  3. Simpan file

  5. Reboot

Jika mengalami kendala dengan mode numpad: Tekan dua kali F6 atau fn + F6.

Aktifkan konfigurasi koneksi cepat Bluetooth:

Jika keyboard membutuhkan waktu terlalu lama untuk terhubung ke komputer melalui Bluetooth (misalnya, ketika kita menekan tombol dan mengaktifkan dari mode sleep), kita dapat mengaktifkan koneksi cepat Bluetooth. Ini biasanya membuat keyboard terhubung dalam waktu kurang dari 1 detik.

Beberapa pengguna melaporkan efek samping masalah pada headphone Bluetooth seperti audio yang muncul dan ketidakstabilan, tetapi saya belum mengalami hal seperti itu.
